# 介绍

逻辑参考《高岛断易》，六爻动画部分参考[biangua](https://github.com/Ovilia/biangua)

# 安装

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```
